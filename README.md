# 餐廳清單

## 介紹
紀錄喜愛的餐廳清單

## 功能
1. 查看所有清單
2. 可瀏覽餐廳的詳細資訊
3. 搜尋特定餐廳

## 使用
1. 確定已安裝 node.js 與 npm
2. 將專案 clone 至本地
```
git clone https://github.com/bst0529/Restaurant-List.git
```
3. 終端機執行出使命令
```
npm install
```
4. 啟動專案
```
npm run start
```
5. 出現以下資訊即可以瀏覽器開啟
```
Web is starting on http://localhost:3000
```

6. 停止執行請在終端機中執行以下命令
```
ctrl + c
```
